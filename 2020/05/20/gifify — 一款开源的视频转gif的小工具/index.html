gifify — 一款开源的视频转gif的小工具

#背景
产品日常工作中，竞品分析和监控是一个重要组成部分。目前我会经常使用GIF图制作竞品示意图。
GIF图相比JPG/PNG等静态图能清晰看到产品的交互和跳转流程，相比视频则具有自动播放的优势，且更容易插入到wiki和PPT中。
但如何将视频转为GIF，其实是一个难题。目前主要有三种途径：

分类 优点 缺点
使用PS 可视化界面，操作简单 PS收费，启动麻烦大材小用
使用在线小工具 query=视频转gif
启动方便		 稳定靠谱的在线小工具比较难找
使用离线小工具 启动简单，自定义选项多 命令行界面，对

本次要介绍的，就是我在Github上找到的一个开源小工具gifify。

#简介
gifify是一款视频转换成压缩Gif动画图片的开源工具。支持任何格式的视频以及选取某个时间段转换。
gifify遵守ISC（基本等同于BSD-clause2）开源协议。

#特征

  • command line interface。命令行界面
  • programmatic JavaScript (Node.JS) stream interface。程序化JavaScript（Node.JS）流接口
  • unix friendly, supports stdin & stdout。unix友好，支持stdin＆stdout
  • optimized! uses pornel/giflossy to generate light GIFS。优化！使用pornel / giflossy生成轻型GIFS
  • lots of options: movie speed, fps, colors, compression, resize, reverse, from & to, subtitles。很多选项：电影速度，fps，颜色，压缩，调整大小，反转，从和到，字幕
  • no temp files used, everything happens in memory。没有使用临时文件，一切都发生在内存中
  • fast! Extracting a 5-second GIF from the middle of a 2-hour movie takes less than 20 seconds。快速！从2小时电影中间提取5秒GIF只需不到20秒

依赖

在安装gifify前，需要安装以下的包:

  • Node.js ($ brew install node)
  • FFmpeg 🐓🐓🐓🐓 ($ brew install ffmpeg)
  • convert, the famous ImageMagick ($ brew install imagemagick)
  • pornel/giflossy, it’s a gifsicle fork (waiting for gifsicle#16 to be merged) ($ brew install giflossy)

You can also use the gifify Docker image which comes with everything installed.
